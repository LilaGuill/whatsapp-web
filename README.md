## Running the project

install Meteor:

```
https://www.meteor.com/
```

Clone this repository :

```
git clone https://github.com/LilaGuill/deliveroo-frontend.git
cd deliveroo-frontend
```

Install packages :

```
meteor npm install
```

When installation is complete, run the project with:

```
meteor run dev
```
